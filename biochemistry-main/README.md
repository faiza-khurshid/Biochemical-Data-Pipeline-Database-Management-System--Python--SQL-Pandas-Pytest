# Title
Biochemistry

## Description
 

 This project is a bioinformatics data processing and analysis pipeline. It allows users to ingest biological datasets, store and query data efficiently, and perform computational analyses. The codebase is structured for modular testing using pytest, ensuring reliability and reproducibility. Designed for flexibility, it can handle multiple datasets, support complex database queries, and serve as a foundation for bioinformatics workflows or research projects.
## Features
Download data from URL and convert to readable database
Query for values in each data category and return filtered dataset
Support for data with multiple values

## Data

The following data is stored in the database:

ID
Substracts
Products
EC number
Enzyme name

## Usage
Can be used by researchers working with Biochemistry Database.
